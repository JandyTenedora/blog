About

Jandy is a senior data engineer based in the UK.

This blog documents a return to first principles — building systems from scratch, working through the foundational texts, and rediscovering the kind of engagement with computing that used to excite him most. It’s easy to spend years operating at the level of abstraction the job demands. This is the deliberate step back down.

What he writes about #

  • deep-systems projects — HTTP servers, Redis clones, SQLite implementations, DNS resolvers. Each one built to understand a layer of infrastructure he uses but doesn’t fully understand yet.
  • First-principles reading — notes from CS:APP, Kurose, SICP. The books that most working engineers skip.
  • Data engineering — pipelines, orchestration, warehousing, stream processing.
  • German — He’s learning German. Occasionally he’ll write about that too.

The setup #

The underlying system is an Obsidian vault running as a personal learning OS, with Claude Code handling daily planning and note generation. He wrote about how it works in the first post.
