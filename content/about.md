I'm Jandy — a data engineer based in the UK, currently in a deliberate upskilling phase.

This blog documents the work: building systems from scratch to understand the internals, working through first-principles computer science books, and writing about what I learn.

## What I write about

- **deep-systems projects** — HTTP servers, Redis clones, SQLite implementations, DNS resolvers. Each one built to understand a layer of infrastructure I use but don't fully understand yet.
- **First-principles reading** — notes from CS:APP, Kurose, SICP. The books that most working engineers skip.
- **Data engineering** — the day job: pipelines, orchestration, warehousing, stream processing.
- **German** — I'm learning German. Occasionally I'll write about that too.

## The setup

The underlying system is an Obsidian vault running as a personal learning OS, with Claude Code handling daily planning and note generation. I wrote about how it works in the [first post](/posts/building-my-learning-os/).
